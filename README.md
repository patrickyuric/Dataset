# Dados

Repositório de Dados para uso em Aulas

Este repositório foi criado com o objetivo de disponibilizar conjuntos de dados em diversos formatos para utilização em atividades práticas em sala de aula.

Os arquivos aqui presentes podem ser utilizados para exercícios de análise de dados, modelagem de banco de dados, manipulação com linguagens de programação (como Python, PHP e JavaScript), consultas SQL, testes de APIs, importação em planilhas eletrônicas e outras aplicações acadêmicas.

Os dados estão organizados em diferentes formatos (como CSV, JSON, XML e SQL), permitindo que os alunos compreendam as particularidades de cada estrutura e pratiquem a transformação, validação e processamento das informações.

⚠️ Importante: Todos os dados disponibilizados neste repositório são fictícios ou provenientes de fontes públicas oficiais. Não há armazenamento ou divulgação de dados pessoais sensíveis ou privados. O objetivo é exclusivamente educacional e acadêmico.

O foco deste repositório é proporcionar aprendizado prático, estimulando a aplicação dos conceitos teóricos em cenários próximos à realidade profissional.
